This is a To Do List Application made using Electron js

This application allows users to add and remove items or tasks from the list allowing users to organize and prioritize tasks/ items

  • File -> Add Item or Ctrl+N to add new items to the list
  • File -> Clear Items or Ctrl+X to clear all items from the list
  • Double click on a single item to delete an individual item from the list
  • File -> About or Ctrl+I to navigate to the About page
  • File -> Quit or Ctrl+Q to quit the application
